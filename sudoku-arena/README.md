# 🧙‍♂️ Sudoku Dungeon (數獨競技場)

一個結合了 **復古像素風 RPG** 與 **數獨解密** 的獨特 Roguelike 地牢冒險遊戲！
玩家必須透過解開數獨謎題來擊敗守關怪物與強大的 Boss、觸發隨機事件、召喚傳奇英雄並施放強大道具。

---

## 🎮 遊戲特色

*   **🧩 數獨地牢探索**
    *   棋盤中隱藏了各種地牢事件，包括 **寶箱 🎁**、**小怪 👾**、**陷阱 ⚠️**、**傳送門 🌀**、**祭壇 🔮** 以及 **Boss 弱點 ✨**。
    *   每填入一個正確的數字即可觸發或解除對應格子上的事件。填錯數字則會扣除玩家的 HP！

*   **🎒 戰術道具背包**
    *   提供多種特色道具（如：*透視放大鏡 🔍*、*炸彈 💣*、*生命藥水 🧪*、*淨化聖水 ✨*）。
    *   極簡化一排式的快捷卡片設計，滑鼠 Hover 或是點選道具後，下方會動態顯示當前持有數量與精細的效果描述。

*   **🧙‍♂️ 英雄召喚與技能系統**
    *   整合精美的抽卡召喚模組，支援「單抽」或「十連抽」！
    *   抽卡可解鎖多達 10 位不同稀有度（R / SR / SSR）與元素的英雄（如 *預言家*、*狂戰士*、*龍騎士* 等）。
    *   玩家清單點選任意解鎖英雄，可**展開精美的手風琴動畫**，查看其專屬的主動/被動能力與戰術說明。

*   **👿 史詩級 Boss 戰挑戰**
    *   挑戰強大 Boss（如：*黑龍法王*），Boss 擁有獨立的 HP 以及特殊的 **「護盾 ⭐」** 機制。
    *   Boss 會對特定的數字施放 **「詛咒 ⚠️」**！如果在詛咒生效時填入該數字，將會面臨嚴厲的 HP 扣減懲罰。
    *   解開「弱點 ✨」格子可對 Boss 護盾進行 Break 削弱！

*   **📜 冒險日誌 Console & 成就圖鑑**
    *   日誌 Console 採用實時降序排版，最新的戰鬥訊息永遠固定在最上方，配合捲軸滾動與像素彩色標示，一目了然。
    *   獨立的成就系統可供切換，隨時檢視自己在技術、進度或收集面解鎖的各項榮譽獎章。

## 🛠️ 技術架構

*   **前端 (Frontend)**
    *   **框架**：Vue 3 (`<script setup>` 組合式 API)
    *   **構建工具**：Vite
    *   **樣式 (Styling)**：Vanilla CSS (精雕細琢的 3D 像素立體像素藝術邊框、玻璃磨砂質感、與動態 CSS Transition 展開效果)

*   **後端 (Backend)**
    *   **框架**：FastAPI (Python)
    *   **ORM**：SQLAlchemy (輕量高效)
    *   **資料庫**：SQLite (內含啟動自動化建表與 Seed 初始化數據)

## ⚡ 快速啟動指南 (透過 Docker)

本專案已完成容器化建置，您可以透過單一指令一鍵運行前後端所有服務：

1.  **確保已安裝 Docker 與 Docker Compose**
2.  **在專案根目錄下執行一鍵啟動：**
    ```bash
    docker compose up -d --build
    ```
3.  **訪問服務：**
    *   🎮 **遊戲前端**：`http://localhost:5173`
    *   ⚙️ **後端 API**：`http://localhost:8000`
    *   📖 **API 自動化文檔 (Swagger)**：`http://localhost:8000/docs`

*(注意：後端啟動時，若發現資料庫 `sudoku.db` 不存在，會自動重新建置並為您載入初始道具、英雄、成就等種子數據。)*
